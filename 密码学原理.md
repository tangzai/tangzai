## 仿射密码

### 加解密公式

```
加密公式：c=Ea,b(m)≡am+b(mod26)
解密公式：m=Da,b(c)≡a-1(c-b)(mod26)
```

## RSA加密算法

解题流程：

1. 任意取两个质数`p,q`
2. 将这两个质数相乘得`n`：`n=p*q`
3. 计算`Ωn`：`Ωn = (p-1) * (q-1)`
4. 选取公钥e，范围：`1 < e < Ωn`
5. 计算私钥d：`ex + Ωy = 1`，d为`e`和`Ωn`的逆元，`Ωn`为被模数
6. 加密公式：`c = m^e mod n`
7. 解密公式：`m = c^d mod n`

### 例题一

**假设需要加密的明文信息为m=85，选择：e=7，p=11，q=13，说明使用RSA算法的加密和解密**

```
p=11
q=13
e=7

n = p*q = 143
fain = (p-1) * (q-1) = 120

d = 7x + 120y = 1 = -17 (求7 和 20 的逆元)

m=85
c= 85^7 mod 143 = 123
```
